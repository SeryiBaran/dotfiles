Справка по горячим клавишам Kitty
-----
F2 - создать новое окно
F3 - предыдущее окно
F4 - следуещее окно
F5 - разделить окно по горизонтали
F6 - разделить окно по вертикали
F8 - закрыть панель (если панель 1 - окно)
Ctrl+left - сузить панель
Ctrl+right - расширить панель
Ctrl+top - расширить панель по вертикали
Ctrl+bottom - сузить панель по вертикали
